SORESPO, short for SOmewhat REalistic Service Provider Orchestrator, is the
primary on-ramp for learning StratoWeave. It is a complete sample
implementation of a StratoWeave-based network automation system, not a toy example.

Use the guides below to start a working lab, load declarative intent, inspect the
layered outputs, and then modify the transforms yourself. These on-site tutorials are
adapted from the upstream SORESPO documentation so the learning path stays inside
stratoweave.org while still pointing back to the source repository where useful.

If you want the upstream project context first, see the
[SORESPO repository](https://github.com/stratoweave/sorespo) and its
[README](https://github.com/stratoweave/sorespo/blob/main/README.md).
