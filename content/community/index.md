# GitHub

StratoWeave is maintained on [GitHub](https://github.com/stratoweave/).
We welcome contributions from the community, and you can get involved by reviewing the
source code, submitting issues, or contributing improvements. Writing documentation,
sharing deployment patterns, and providing feedback are also valuable ways to participate.

# Zulip chat

The StratoWeave community uses the [LF Networking Zulip](https://linuxfoundation.zulipchat.com/)
chat platform for discussions, questions, and collaboration.
